Most editors support CTRL + SHIFT + M to preview any .md file.

### Font / Header Sizing Using '#'
```
# H1 - large banner size
## H2 - small banner size
### H3 - large header
#### H4 - small header
##### H5 - large bold text
###### H6 - bold text
```

# H1 - large banner size
## H2 - small banner size
### H3 - large header
#### H4 - small header
##### H5 - large bold text
###### H6 - bold text

```
